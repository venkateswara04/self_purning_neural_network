# Self-Pruning Neural Network - Results Report

## Why L1 on sigmoid gates encourages sparsity

SparsityLoss = mean( sigmoid(gate_score_i) ) for all i

The L1 norm has a constant gradient of 1 regardless of gate size.
This keeps pushing gate values all the way to zero (unlike L2 which
slows down near zero and never fully zeroes anything).

Classification loss pushes gate_scores positive for useful weights.
Sparsity loss pushes ALL gate_scores negative.
Result: useful weights survive (gate near 1), useless ones are pruned (gate near 0).
